# Online Medical Appointment Booking System

This is the final database project for the AUCA Database Course.  
The goal of the project is to design and implement a complete database system using PostgreSQL.

## Project Components
- ER-diagram
- Normalized database schema (3NF)
- PostgreSQL implementation (CREATE TABLE scripts)
- Basic SQL queries (SELECT, INSERT, UPDATE, DELETE)
- Advanced SQL queries (JOIN, GROUP BY, subqueries, views, triggers)
- Demonstration of transactions
- Demonstration of indexing
- Backup and restore strategy
